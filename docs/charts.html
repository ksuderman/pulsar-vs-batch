Galaxy-Reported CPU Slots: Batch vs Pulsar

Batch explicitly sets GALAXY_SLOTS in container_script.sh. Pulsar relies on CLUSTER_SLOTS_STATEMENT.sh fallback (reports 1, but VM has correct cores via right-sizing).

Runtime Ratio vs Batch Slot Count

If Pulsar tools were truly limited to 1 thread, the overhead % would roughly match the slot count. Actual ratios are lower, confirming tools partially detect available cores (2GB data).
